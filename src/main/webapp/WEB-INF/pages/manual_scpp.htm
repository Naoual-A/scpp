1.    Acceso a la aplicación

Al igual que cualquier otro servicio web, el acceso a scpp ha de realizarse mediante la introducción de la url en un navegador web. Una vez introducida la dirección, el sistema nos muestra un formulario de autenticación como el de la siguiente captura, mediante el cual, el usuario ha de identificarse introduciendo su Id y su contraseña.

Una vez identificado, el sistema muestra la siguiente pantalla de bienvenida:

 

1.1.                   Darse de alta como padre

Si el usuario desea darse de alta como padre, ha de dirigirse a la páginca de autenticación y pulsar el botón Dar de alta.

Cuando pulsa el botón rojo, el sistema muestra un formulario con los datos que hay que rellenar. Es importante disponer del Id del alumno al que se va a vincular el id del padre.

El usuario ha de rellenar todos los campos para poder darse de alta como padre en el sistema. Una vez rellenados todos los campos, el usuario debe pulsar el botón verde. En caso de que no quiere seguir con la operación, debe pulsar el botón Cancelar.

Si el usuario pulsa el botón Dar de alta, y todos los campos introducidos son correctos, el sistema lo redirige a la pantalla de autenticación con el siguiente mensaje.

Una vez en esta pantalla, el usuario puede introducir sus datos y conectarse al sistema.

1.2.                   Consultar el perfil de usuario

Una vez dentro del sistema, el usuario puede consultar su información personal de la que dispone el sistema. Para ello, ha de dirigirse mediante en el menú lateral Perfil de usuario.

1.3.                   Consultar perfil de alumno vinculado

En el caso de un usuario padre, se puede consultar el perfil del alumno vinculado. Para ello, en la pantalla de perfil de usuario, dispondrá de un link que lo redirigirá al perfil vinculado.

2.    Módulo de comunicación padres-profesores

Este módulo engloba las siguientes opciones:

·         Enviar un nuevo mensaje.

·         Leer un mensaje.

·         Responder a un mensaje recibido.

·         Consultar mensajes recibidos.

·         Consultar mensajes enviados

2.1.                   Enviar un nuevo mensaje

Para enviar un nuevo mensaje, el usuario ha de dirigirse a la pantalla de nuevo mensaje mediante el menú lateral Mensajes -> Nuevo.

El sistema nos muestra el formulario de la captura anterior. Para enviar el mensaje hay que, selecciona run destinatarios de los que el sistema nos muestra, introducir un asunto, introducir el mensaje que deseamos enviar, y a continuación pulsar el botón Enviar.

Si deseamos cancelar la operación, pulsamos el botón Cancelar.

 

2.2.                   Leer un mensaje

Para leer un mensaje el usuario ha de derigirse a la pantalla de mensajes recibidos/enviados mediante el menu Mensajes->Mensajes recibidos/enviados.

Una vez nos encontramos en esa pantalla, el usuario escoge el mensaje que desea leer, y pulsa el boton Leer. A continuación, el sistema muestra una nueva pantalla con el contenido del mensaje.

2.3.                   Responder a un mensaje recibido

Para responder a un mensaje, estando en la pantalla anterior (Leer mensaje), el usuario pulsa el botón de Responder, y el sistema mostrara un nuevo formulario para escribir la respuesta del mensaje.

2.4.                   Consultar mensajes recibidos

Para consultar los mensajes recibidos, hay que pulsar en el menú lateral en Mensajes -> Mensajes recibidos.

2.5.                   Consultar mensajes enviados

Para consultar los mensajes enviados, hay que pulsar en el menú lateral en Mensajes -> Mensajes enviados.

4.    Gestión de tareas

Este apartado engloba las siguientes tareas:

·         Consultar tareas.

·         Crear tarea.

·         Editar tarea.

·         Consultar tarea.

·         Eliminar tarea.

4.1.                   Consultar tareas

Para consultar el conjunto de tareas que tiene disponible cada usuario, hay que dirigirse mediante el menú lateral Tareas -> Mis tareas, a la siguiente pantalla:

4.2.                   Crear tarea

Para crear una nueva tarea, lo podemos hacer o mediante el botón de Nueva tarea que encontramos arriba del listado de tareas, o accediendo mediante el menú lateral Tareas -> Nueva tarea.

El usuario escoge la asignatura para la cual está creando la tarea, a continuación introduce el título, la descripción, una fecha de inicio y una fecha de finalización. Una vez rellenados todos los campos, pulsar el botón Crear.

4.3.                   Editar tarea

Para editar una tarea, en el listado de tareas (Tareas->Mis tareas) localizamos la tarea que deseamos editar, y pulsamos en el botón editar. Una vez pulsado el botón, se mostrara la página para editar la tarea.

Editamos los campos deseados, y a continuación pulsamos el botón Guardar cambios.

4.4.                   Consultar tarea

Para consultar una tarea, al igual que en el caso anterior, nos dirigimos al listado de tareas (Tareas->Mis tareas) localizamos la tarea, pulsamos el botón Ver, y se mostrará el contenido de la tarea.

4.5.                   Eliminar tarea

Para eliminar una tarea, nos dirigimos al listado de tareas (Tareas->Mis tareas) localizamos la tarea que deseamos eliminar, pulsamos el botón Eliminar, y se nos mostrará un mensaje de confirmación.

Al pulsar Aceptar, la tarea queda eliminada.

Siempre podemos cancelar la operación pulsando el botón Cancelar.

Siempre podemos cancelar la operación pulsando el botón Cancelar.
